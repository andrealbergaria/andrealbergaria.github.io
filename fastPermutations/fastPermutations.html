Python Implementation C Implementation

The objective is to permute all elements in a list or array (using python list or/and c arrays).

In python, we just rotate the list, with line lst = lst[1:] + lstl[:1]..and keep iterating ...

In c language, we use the algorithm:

  1. Save first byte.
  2. For each byte, swamp values of next byte with before byte.
  3. Set saved byte to first position

Example

  1. array[i] = 'a' , 'b', 'c'.
  2. Save 'a'.
  3. Swamp value of 'a' with 'b'.
  4. swamp value of 'b' with 'c'....etc
  5. put first byte on the end position('c' position)
    6. keep iterating until all permutations are used..
